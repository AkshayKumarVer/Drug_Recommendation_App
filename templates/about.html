About

The Drug Recommendation System is a cutting-edge application designed to provide insights and potential solutions for patients based on the symptoms they describe. Using advanced data and algorithms, this system identifies the most suitable drugs for specific conditions, enabling faster and more accurate healthcare decisions.

Technology Stack

  • Large Language Model (LLM): facebook/bart-large-mnli model is leveraged for its ability to perform Natural Language Inference(NLI)
  • Framework: Flask is used as the backend framework to manage requests, routing, and data processing efficiently.
  • Database: A structured database containing drug information, conditions, and recommendations powers the system's insights.
  • Frontend: HTML, CSS, and JavaScript for a user-friendly interface and seamless navigation.

Project Highlights

  • Integrated symptom-based drug recommendation powered by natural language processing (NLP).
  • Utilized LLMs to analyze and interpret medical text data for accurate predictions.
  • Built a robust data pipeline to preprocess symptom data and align it with drug-condition mappings.
  • Designed a responsive web interface for better user experience.
  • Implemented model deployment strategies to integrate the LLM with the Flask application.

Please note that this tool is for informational purposes and should not replace professional medical advice.
